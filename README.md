## Développement

```bash
npm install                 # installer les dépendances
npm run dev -- --open       # lancer le serveur de développement
npm run build               # compiler pour la production (output dans dist/)
npm run preview -- --open   # prévisualiser le build de production localement
```

## Qualité du code

```bash
npm run format     # reformater automatiquement tous les fichiers
npm run lint       # vérifier le formatage (prettier) et les erreurs (eslint)
npm run check      # vérification des types TypeScript et Svelte
```

## Mettre à jour les dépendances

```bash
npm update
```
